Available DGTE Clients:

  • Chess (ZIP file) - uses EPD econoding (written in Java) - works with the Chenard Chess plugin.
  • Mancala (ZIP file) - text-based Mancala client (written in Java) - works with the Mancala plugin.
  • Primes (ZIP file) - text-based prime-number checker (written in Java) - works with the Primes 2000 plugin.
